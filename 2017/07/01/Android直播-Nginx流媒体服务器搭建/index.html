二、配置

安装成功后，我们可以根据以下命令来查看nginx安装的一些信息

  • nginx服务器根目录所在位置：/usr/local/var/www
  • nginx配置文件所在位置：/usr/local/etc/nginx/nginx.conf
  • ngnix -s reload：重新加载配置文件
  • ngnix -s reopen：重新打开日志文件
  • ngnix -s stop：停止服务器
  • ngnix -s quit：退出服务器
