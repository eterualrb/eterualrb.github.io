2D绘图基础

2016.03.28 王白告

参考：Android群英传

1. 基本概念

Canvas是Android 2D图形编程中一个很常用的概念，通常由3个基本的绘图组件组成:

  • Canvas “画布”，提供了绘图方法，可以向底层的位图绘制基本图形。
  • Paint “画笔” 指定如何将基本图形绘制到位图上
  • Bitmap 绘图的表面

2. 常用API

系统通过提供Canvas对象来提供绘图方法。它提供了各种绘制图像的API:

Canvas API 功能
drawPoint() 绘制点
drawLine() 绘制线
drawRect() 绘制矩形
drawVertices() 绘制多边形
drawArc() 绘制弧
drawCircle() 绘制圆

此外Paint作为一个非常重要的元素，功能也是很强大的，以下简单的列举了一些它的属性和对应的功能:

Paint API 功能
setAntiAlias() 设置画笔的锯齿效果
setColor() 设置画笔的颜色
setARGB() 设置画笔的A,R,G,B值
setAlpha() 设置画笔的Alpha值
setTextSize() 设置字体尺寸
setStyle() 设置画笔的风格(空心或实心)
setStrokeWidth() 设置空心边框的宽度

正是由于画笔功能的不同，再结合各种不同绘图的API就可以实现不同的绘图效果。
