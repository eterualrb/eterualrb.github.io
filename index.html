MPEG-4白皮书

发表于 2021-08-01

常用白皮书

MPEG-4是一组协议族，其编号为ISO/IEC 14496，常用的有

Part Number Title Description
1 ISO/IEC 14996-1 System MPEG-4 的复用、同步等系统级特性
2 ISO/IEC 14996-2 Video 视频压缩标准
3 ISO/IEC 14996-3 Audio 音频压缩标准
10 ISO/IEC 14996-10 Advanced Video Coding (AVC) 先进视频编码标准，即H.264/AVC标准
12 ISO/IEC 14996-12 ISO base media format ISO规定的基本文件封装容器格式标准
14 ISO/IEC 14996-14 MP4 file format 第12部分的扩展，定义MP4封转格式
15 ISO/IEC 14996-15 Advanced Video Coding(AVC) file format 第14部分的扩展，规定了保存H.264/AVC标准视频容器格式

Box

MP4文件的基本结构为Box(也称作Atom)，ISO/IEC 14496-12中定义了两种Box类型：Box和FullBox

  • size: box大小，包括header和body。当size=1时使用largesize字段存放box大小(mdat类型)；当size=0时，代表文件结尾

  • type: box类型， 直接取ASCII码(box字节序为大端字节序)。如果为uuid，代表用户扩展类型
